## Usage

1. Open `index.html` in your web browser
2. Click the play button to start a work session
3. The timer will automatically switch between work and break periods
4. Customize settings by clicking the gear icon
5. Track your completed sessions

## Keyboard Shortcuts

- `Space`: Start/Pause timer

## Settings

- **Work Duration**: Length of work sessions (default: 25 minutes)
- **Short Break Duration**: Length of short breaks (default: 5 minutes)
- **Long Break Duration**: Length of long breaks (default: 15 minutes)
- **Long Break After**: Number of sessions before long break (default: 4)
- **Enable Sound Notifications**: Toggle audio alerts
- **Auto-start Breaks**: Automatically start break timers
- **Auto-start Pomodoros**: Automatically start work sessions

## Installation

Simply download all files to a folder and open `index.html` in your browser. No build process or dependencies required!

## The Pomodoro Technique

1. Choose a task to work on
2. Set the timer for 25 minutes
3. Work until the timer rings
4. Take a short 5-minute break
5. After 4 pomodoros, take a longer 15-minute break
